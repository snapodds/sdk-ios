SnapscreenSnapUIConfiguration

Configuration for the customizing the Snap UI. See https://docs.snapodds.com/docs/mobile-sdk/ios/snapodds-operator/customization or https://docs.snapodds.com/docs/mobile-sdk/ios/snapodds-sport-media/customization for customization options

  • hideDefaultViewFinderAndQuadrangleDetection

    Hides the view finder UI

  • snapTitle

    Snap Title - set to nil if you want to customize this on your own

  • snapHintText

    Snap Hint/Instruction shown in the snap ui

  • showHelpButtonText

    Show Help Button Text - shown as the right side bar button item in the navigation bar when the snapping UI is visible

  • helpTitle

    Help Title

  • snapProgressText

    Progress text while manual snap is being handled

  • snapErrorGeneralText

    Error text when any error occured during snapping

  • snapErrorConnectionIssueText

    Error text when any connection error occured during snapping

  • snapErrorTechnicalIssueText

    Error text when a technical issues error occured during snapping

  • snapErrorNoResultsText

    Error text when no result was found for manual snap

  • hidePoweredBySnapOddsBranding
