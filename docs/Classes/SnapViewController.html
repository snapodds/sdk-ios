SnapViewController

public class SnapViewController : UIViewController
extension SnapViewController: AVCaptureVideoDataOutputSampleBufferDelegate

SnapViewController offers the snapping UI including all the logic to your application. Instantiate directly by passing the desired SnapConfiguration and set an appropriate SnapscreenSnapDelegate to receive callbacks.

It is recommended to present the SnapViewController modally in a UINavigationController. Note: Please note that you need to explicitly add a navigation item for dismissing the ViewController.
